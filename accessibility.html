Accessibility Statement

Last updated: 24 July 2026

This is an accessibility statement from Scott Labz.

Measures to Support Accessibility

Scott Labz takes the following measures to ensure accessibility across scottlabz.com:

  • Include accessibility throughout our internal policies.
  • Assign clear accessibility goals and responsibilities.
  • Employ formal accessibility quality assurance methods.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

scottlabz.com is partially conformant with WCAG 2.1 Level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard without exceptions.

Additional Accessibility Considerations

Although our goal is WCAG 2.1 Level AA conformance, we have also incorporated the following additional accessibility practices:

  • Motion Sensitivity: Respecting the prefers-reduced-motion media query to suppress unnecessary background animations for users with vestibular spectrum disorders.
  • Enhanced Keyboard Navigation: Providing distinct visual focus indicators and logical tab orders for keyboard-only users.
  • Form State Protection: Designing forms to prevent loss of entered data in the event of timed sessions or accidental page refreshes.
  • Continuous Testing: Regularly auditing code using automated accessibility testing suites (axe-core) alongside manual screen reader checks.
